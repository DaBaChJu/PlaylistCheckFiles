Programme qui vérifie si tous les fichiers d'une playlist existent bien.
Peut chercher si des fichiers ont été déplacés, supprimés, ou pourraient
manquer à une playlist (en supposant que les playlists traitées contiennent
habituellement tous les éléments des dossiers inclus dans ces listes).
Utilise des couleurs et des info-bulles pour marquer les cas, et peut
afficher "All Green" dans le titre si tout est vert sans problème.

Program that check if all files within a playlist indeed exist.
Can check if files were moved, deleted, or could be missing from
a playlist (assuming the dealt with playlists usually contain all
element within the folders included in the lists).
Use colors and tooltips to describe the cases, and can display
"All Green" in the title bar if everything is green and ok.
